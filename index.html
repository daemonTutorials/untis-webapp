untis-webapp

This WebApp is currently available in german and english (US) language. The preconfigured server is the server of my school in Germany, Lower Saxony. A running version can be accessed at my school Dev-Server.

The backend runs on node.js and that loads the required data from the school web server, because the school web server doesn't send CORS-Headers. The backend isn't published yet.

These tasks are currently in the pipe:

  • Load the teacher data from the school web server (get teacher access for that).
  • Allow the configuration of the URL for receiving data with configuration JSON-Object of the Vertretungsplan class.
