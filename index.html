untis-webapp

This WebApp is currently available in german language for students of the BBS Lingen in Germany, Lower Saxony. A running version can be accessed at my school Dev-Server.

At the moment it depends on a php-file that is a workaround to bypass the missing CORS-Header of the school server. The server backend is now hosted on openshift.com with Node.js.

UNTIS Webapp is maintained by daemonTutorials. This page was generated by GitHub Pages using the Cayman theme by Jason Long.
